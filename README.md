# 📰 NewsPulse – AI News Intelligence Platform

An intelligent news analysis platform that fetches, processes, and analyzes news using NLP techniques.

---

## 🚀 Features

* 📰 Fetch latest news using APIs
* 🧠 NLP-based sentiment analysis
* 📊 Trend detection & keyword extraction
* 📈 Data visualization dashboard
* 🔐 Secure authentication system

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Database:** SQLite
* **Libraries:** Pandas, Matplotlib, TextBlob, Scikit-learn

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/lakshmi007singh-ui/newspulsee.git
cd newspulsee
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Setup environment variables

Create a `.env` file and add:

```
GOOGLE_CLIENT_ID=your_client_id
GOOGLE_CLIENT_SECRET=your_secret
```

---

## ▶️ Run the Project

```bash
streamlit run news_pulse.py
```

---

## 🔐 Security Note

* `.env` file is ignored for security
* Do NOT upload API keys
* Use `.env.example` as reference
